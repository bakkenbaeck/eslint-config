# Use these rules in your project

## Installation
Add @bakkenbaeck/eslint-config as a dependency: 

### yarn:

`$ yarn add eslint @bakkenbaeck/eslint-config --dev`

### npm:  

`$ npm i eslint @bakkenbaeck/eslint-config --dev`


## Configuration
### `package.json`
````
"eslintConfig": {
  "extends": "@bakkenbaeck/eslint-config"
},
````

### `.eslintrc`
```
{
  extends: ["@bakkenbaeck/eslint-config"]
}
````
